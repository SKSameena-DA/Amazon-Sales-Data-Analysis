# Amazon Sales Data Analysis

## Project Overview
This project was completed as part of the TechTip24 Workshop. The objective was to analyze Amazon sales data and create interactive dashboards to gain valuable business insights. The project focuses on understanding sales performance, customer purchasing patterns, and product trends using data analytics techniques.

## Objectives
- Analyze Amazon sales data.
- Clean and preprocess the dataset.
- Identify sales trends and revenue patterns.
- Discover top-performing products and categories.
- Create interactive dashboards for better decision-making.

## Tools Used
- Power BI
- Microsoft Excel

## Dataset
The dataset contains Amazon sales information, including:
- Product Category
- Product Name
- Sales Amount
- Quantity Sold
- Customer Details
- Order Date
- Region

## Key Analysis
- Sales trend analysis over time
- Top-selling products and categories
- Revenue analysis
- Regional sales performance
- Customer purchasing behavior
- Interactive filtering using slicers

## Dashboard Features
- KPI Cards
- Bar Charts
- Line Charts
- Pie Charts
- Slicers and Filters
- Sales Summary

## Skills Gained
- Data Cleaning
- Data Visualization
- Dashboard Development
- Business Intelligence
- Sales Data Analysis
- Report Design

## Conclusion
This project enhanced my understanding of data analytics by transforming raw sales data into meaningful visualizations and actionable insights. It strengthened my Power BI and Excel skills while improving my ability to analyze business data effectively.
